## Sobre mí

Desarrollador enfocado en sistemas geoespaciales y en la construcción de Infraestructuras de Datos Espaciales (IDE) utilizando tecnologías de código abierto. Trabajo principalmente en el desarrollo de plataformas que integran procesamiento de datos espaciales, servicios geográficos y visualización en mapas.

Actualmente cursando un Magíster en Data Science, con interés en aplicar machine learning a problemas territoriales y ambientales.

***

## Experiencia técnica

- Backend: Python, Django, Django REST Framework, Scikit-learn  
- Geoespacial: PostGIS, GeoServer, GDAL, Rasterio, GeoPandas, pg_tileserv  
- Frontend: Vue.js, Leaflet, MapLibreGL, JavaScript  
- Procesamiento y tareas asíncronas: Celery, RabbitMQ, Redis  
- Infraestructura: Docker, Linux (Ubuntu Server), Nginx, GitHub Actions  
- Base de datos: PostgreSQL
